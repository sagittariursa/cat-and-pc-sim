# Cat and PC Simulator

A simulation of a 7-month-old kitten getting dangerously close to an RTX 4060. 

## How to Play
1. Download the files from this repository.
2. *Requirement:* You must have *Java* installed.
   - [Download Java here](https://www.oracle.com/java/technologies/downloads/#windows)
3. Run cat and pc.bat to start the protocol.
