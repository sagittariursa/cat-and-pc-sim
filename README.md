# Cat and PC Simulator

A simulation of a 7-month-old kitten encountering a high-end PC. 

## How to Play
1. Download the repository as a ZIP (Green *Code* button -> *Download ZIP*).
2. *Extract All* files from the ZIP folder.
3. *IMPORTANT:* You must have Java installed. [Download Java here](https://www.oracle.com/java/technologies/downloads/#windows).
4. Run cat and pc.bat.

## If Windows Blocks the Game:
Windows might show a "Smart App Control" warning because this is an unsigned indie project.
1. Right-click *cat and pc.bat*.
2. Click *Properties*.
3. Check the *Unblock* box at the bottom.
4. Click *OK* and run it again!
